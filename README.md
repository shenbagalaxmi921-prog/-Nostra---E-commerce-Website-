# -Nostra---E-commerce-Website-

Nostra is a responsive front-end e-commerce website built using HTML, CSS, and JavaScript. The project demonstrates basic web development concepts including responsive layout design, DOM manipulation, and interactive product search and filtering.

Features

        Responsive website layout for desktop and mobile devices
        
        Home page with introduction and highlights
        
        Collections page displaying products
        
        Real-time product search functionality using JavaScript
        
        Filter products by category using JavaScript
        
        Mobile friendly hamburger navigation menu
        
        Contact page with a simple contact form

Pages Included

    Home Page – Main landing page
    
    Collections Page – Displays products with search and filter functionality
    
    Contact Page – Allows users to send messages

Technologies Used

    HTML5
    
    CSS3
    
    JavaScript

Project Structure
      Nostra-Ecommerce
      │
      ├── index.html        # Home page
      ├── collections.html  # Products and filters
      ├── contact.html      # Contact page
      ├── style.css         # Website styling
      └── script.js         # Search, filter, and menu functionality
Functionality
      Search Feature

          Users can search products in real time on the collections page. The product list updates dynamically based on the text entered in the search bar.

      Filter Feature

          Products can be filtered by categories such as shirts, hoodies, and t-shirts using JavaScript.

      Mobile Navigation

          A responsive hamburger menu allows users to access navigation links on smaller screens.

      Learning Objectives

            This project helps practice:

                  Building multi-page websites
                  
                  Responsive web design
                  
                  JavaScript DOM manipulation
                  
                  Event handling
                  
                  Implementing search and filter functionality
